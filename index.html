JellyJolly

是一个开源（GPL V3）的、Java编写的、运行在Tomcat服务器上的博客引擎。

Openshift发布版

运行在由红帽（Redhat）提供的PAAS平台Openshift之上，通过rhc命令行接口部署。

获取JellyJolly项目源码

整个项目由若干子项目构成，包括：

  • JellyJolly Model，数据模型
  • JellyJolly Persistence API，数据持久化接口
  • JellyJolly Plugin API，插件接口（尚未完成）
  • JellyJolly Core，博客引擎核心
  • JellyJolly Persistence Mysql Implementation，数据持久的MySQL参考实现
  • JellyJolly Openshift Distribution，Openshift发布版

通过访问各子项目主页，根据页内所示git仓库地址，来单独获取某个子项目源码 例如，可以通过

git clone https://github.com/JellyJollyTeam/JellyJollyOpenshiftDistribution.git

获取Openshift发布版的源码。 当然无需获取所有的子项目来构建整个项目，JellyJolly使用Maven构建工具，可以通过

mvn clean install

来解决其它并未下载源码的子项目所依赖的工件。

定制自己的JellyJolly发布版

JellyJolly不仅仅运行在Openshift平台上。她可以运行在您配置的Tomcat + MySQL服务器上。 您无需为了定制去学习开发相关的知识。当然，了解一些Java、数据库或者Web开发的相关知识都会帮助您更好地定制自己的发布版。
